# First-Map-project
 Interactive map of German automotive headquarters (Porsche &amp; BMW) built with Python and Folium.
[CLICK HERE TO OPEN THE LIVE MAP](https://dunkmeistergg.github.io/First-Map-project/First_map_code.html)
## 🛠️ Tech Stack
* **Language:** Python 3
* **Library:** Folium
* **Features:** Custom markers, distance lines, and a 50km logistics buffer zone.
